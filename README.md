# Getting Started with Create React App

How to install ?

After clone this repository, in the project directory, first run:

### `npm install`

or this one if have some problems

### `npm install --force`

Then, run:

### `npm start`

If every thing work well, you can find store in:

### `http://localhost:3000/`

Good luck <3
